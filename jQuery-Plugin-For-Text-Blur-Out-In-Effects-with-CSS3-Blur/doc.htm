jQuery Blur

by Chris Cates

jQuery Blur gives your text a nice blur effect. Its simple and its awesome. Its purpose is to be a fancier version of fadeOut and fadeIn.

Setup is easy

You will need jQuery Transit. Since jQuery Animate does not support CSS3 Transforms.

<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript" src="jquery.blur.js"></script>
<script type="text/javascript" src="jquery.transit.js"></script>

Download

jquery.blur.js

jquery.blur.min.js

jquery.transit.js

Blur works in one line and supports function chaining.

First Click me to blur out and in

$('.Example1').click(function() {
	$(this).blurOut(1000).blurIn(2000);
	/*
	There is only one parameter. Which is determines the ms 
	it takes to complete the animation.
	eg: $.blurOut(500); will take 500ms (half a second).
	*/
});

You can also change the default black blur color by using the bColor attribute.

<h3 bColor="green">First Click me to blur out and in</h3>

The Javascript works similar to jquerys hide... However to make the element appear again

$('your element').css({opacity: 1});
/*
	In CSS: it would be the same, opacity: 1;
*/
